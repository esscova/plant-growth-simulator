Simulador de Crescimento de Plantas

Ajuste os sliders para simular o crescimento da planta com base em luz e água.



Sobre

Este projeto aplica o Planejamento de Experimentos para simular o crescimento de plantas, usando luz (0-12 horas) e água (0-100 ml) como fatores. Desenvolvido com Flask (backend), HTML/JavaScript (frontend) e Python (análise estatística).
